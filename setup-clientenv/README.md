# env-for-k8s
set of CLI tool for k8s/docke

# Useful Tools
### [kubectx](https://github.com/ahmetb/kubectx)
- contextの切替

### stern
- 複数のpodのログの閲覧

### dive
- dockerのOverlaFSの中身を全て表示
